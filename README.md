This is task : Insurance Quote Management System API
Steps to run : 

  1. Clone the repository into your local System which is having the intelliJ
  2. Open the project.
  3. Run the WealthVellyTaskApplication by right clicking on it
  4. Open Postman
  5. Select Get httpMethod and write the localhost/api/v1/insurance-quotes/{id} and then call the api
  6. Select Post httpMethod and write the localhost/api/v1/insurance-quotes and in body fill the required data in json format and then call the api
